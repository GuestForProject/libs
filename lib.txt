   Пример (1 *.py)

1. docx
    1) python-docx
    2)  add_heading - добавить заголовок
        add_paragraph - добавить параграф
        save - сохранение файла
        add_table - создать таблицу
        add_picture - добавить картинку
    3) Создание отчётов по продажам
2. xlsx
    1) openpyxl
    2) get_sheet_by_name - получение страницы
       cell(row=1,column=2) - получить ячейку
    3) Построение таблиц по записям БД
3. txt
    1) txt
    2) is_word - это str?
        count_words - количество слов
    3) Негде
3. pptx
    1) python-pptx
    2) add_slide - добавить слайд
    3) Создание призентации для показа графиков выручки
4. odt
    1) odfpy
    2) addElement - добавить элемент
       Style - установить стиль
    3) Негде
5. jpeg
    1) Pillow
    2) load - загрузить в lib
       rotate - поворот
       show - показать изображение
       crop - кропнуть изображение
    3) Обрабатывать изображения чеков
6. png
    1) Pillow
    2) load - загрузить в lib

7. mp3
    1) AudioPlayer
    2) AudioPlayer - относительный путь до файла
       play - воспроизведение
       pause - пауза
       resume - продолжить
       stop - остановить
    3) Sound track для работников
8. mp4
    1) cv2
    2) VideoCapture - входной файла
       rectangle - создание прямоугольника
    3) Рисование прямоугольников
9. ini
    1) configparser
    2) read - прочитать файл
       sections - get секции
       get - получить значение
    3) Конфигурационные данные для подключения к БД
10. md
    1) python_md
    2) h1 - create заголовок
       table - создать таблицу
       write_to_file - сохранить файл
    3) Документации для продукта
11. html
